### Proje Açıklaması

Bu program, kullanıcının girdiği n ve r sayılarına göre kombinasyon hesaplaması yapar. Kullanıcıdan n ve r sayılarını girmesi istenir ve C(n,r) kombinasyonunu hesaplar.

---

### Dosyalar

Bu repo aşağıdaki dosyaları içermektedir:

- `README.md`: Proje hakkında genel bilgiler ve açıklamalar içeren dosya.
- `KombinasyonProgrami.java`: Programın kaynak kodunu içeren Java dosyası.

---

### Nasıl Kullanılır

1. Proje dosyalarını bilgisayarınıza indirin veya kopyalayın.
2. Java IDE'si (Integrated Development Environment) veya metin düzenleyici gibi bir yazılım kullanarak `KombinasyonProgrami.java` dosyasını açın.
3. Kaynak kodunu inceleyin ve gerekirse düzenlemeler yapın.
4. Düzenlemeleri tamamladıktan sonra, `KombinasyonProgrami.java` dosyasını derleyin ve çalıştırın.
5. Program size n ve r sayılarını girmenizi isteyecektir. İstenilen bilgileri girin ve Enter tuşuna basın.
6. Program, girilen n ve r sayılarına göre C(n,r) kombinasyonunu hesaplayıp ekrana yazdıracaktır.
